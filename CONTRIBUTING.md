# Contributing

Thank you for improving TierListMaker.

## Development

- Use Qt 6.5 or newer.
- Keep local paths in `CMakeUserPresets.json`, never in committed files.
- Run formatting before opening a pull request.
- Add focused tests for persistence, project mutation, and asset-path behavior.

## Pull Requests

- Keep changes scoped.
- Explain user-visible behavior changes.
- Include screenshots for UI changes when practical.
- Do not commit imported user images, logs, crash dumps, tokens, certificates, or local build products.
