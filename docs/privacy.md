# Privacy

TierListMaker is local-first.

- The app does not upload user images.
- The app does not upload project files.
- The app does not use telemetry by default.
- Recent-project metadata is stored locally.
- Update checking is a placeholder and remains opt-in if implemented in the future.

Project files may include image filenames and paths. Treat project files as private if your image names or folder names are sensitive.
