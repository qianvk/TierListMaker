# User Guide

## Create a Project

Open TierListMaker. A new unsaved project is created automatically. Use **New** to reset to another project.

## Import Images

Use **Import** or press `Cmd/Ctrl+I`. You can select multiple images. Dropping image files onto the image pool also imports them.

## Rank Images

Drag images from the pool into tier rows. Drag them between rows to move them, or back to the pool to unrank them.

## Edit Rows

Use row controls to rename, recolor, move, add, delete, or reset rows.

## Preview

Select or hover an image and press Space. Press Space or Escape again, or click outside the preview, to close it.

## Save and Export

Projects are saved as `.tlmproject` files. Export creates PNG, JPEG, or PDF output without mutating the project.
